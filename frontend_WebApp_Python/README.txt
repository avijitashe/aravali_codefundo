---

Prerequisite
-------------
1. Install Python 3.7
2. Install flask (pip install flask)
3. Install wtf (pip install flask-wtf)

Now the environment is ready for flask app develoment

Set Up the web app
-------------------
1. go to the project directory and open the command prompt.
   set the following environment variables.
   (i) set FLASK_APP=aravali.py
  (ii) flask run

2. Now open the browser and copy the url http://127.0.0.1:5000/
   This will lead you to the web app.

Use the "Aravali" app
----------------------
1. Register yourself using the register link.
2. Log in to the app with the above credentials.
3. Now upload the data set and train the model.
